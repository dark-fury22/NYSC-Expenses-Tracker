# NYSC Expense Tracker (Personal Finance Web App)

## Description
The NYSC Expense Tracker is a personal finance web app designed specifically for NYSC corps members to manage their allowance, expenses, and overall budget efficiently. The app is built with HTML5, CSS3, and JavaScript, leveraging DOM manipulation, LocalStorage, and interactive charts for a responsive, user-friendly experience.

It helps users track income and expenses, categorize spending, visualize cash flow with horizontal gradient bar charts and animated line charts, and make informed financial decisions during their service year.

Short One-Liner:  
_Track your NYSC allowance and expenses effortlessly with interactive charts and a clean, responsive interface._

---

## Features
- Add and remove income and expenses
- Categorize spending: Allowance, Food, Transport, Bills, Others
- Dynamic total balance display with color-coded feedback
- Horizontal gradient bar chart showing spending by category
- Animated line chart showing income vs expenses over time
- Hover tooltips on charts displaying exact amounts
- Filter transactions by category and date range
- Persistent data storage using LocalStorage
- Fully responsive and mobile-friendly design

---

## Technologies Used
- HTML5 – Web app structure
- CSS3 – Styling, gradients, animations, responsive layout
- JavaScript (ES6) – Logic, DOM manipulation, LocalStorage, chart rendering
